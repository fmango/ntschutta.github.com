--- layout: post meta: {} status: publish tags: - Ajax - Book Reviews - Software title: Pro Ajax and Java Frameworks type: post published: true --- Ryan and I are proud to announce our second collaboration, Pro Ajax and Java Frameworks! We just got our author copies so I expect it will be available shortly from the major retailers. This book largely picks up where Foundations ends: after a general Ajax overview, we cover some common tools (including my new favorite Firebug) then we discuss a number of frameworks (Dojo, Prototype, script.aculo.us, DWR, etc). Once we've covered the client side goodness, we walk readers through integrating Ajax into common Java frameworks including Struts, JSF, Tapestry, and Spring. We've been thrilled with how well received Foundations has been and we hope people learn as much from reading Pro Ajax as we did writing it!
