--- layout: post meta: {} status: publish tags: - Development - Software title: Beautiful Code type: post published: true --- I don't get to go to quite as many conferences as I'd like but luckily more and more organizers are putting talks on-line or releasing interviews and other useful tidbits. YouTube is home to a bunch of Google Tech Talks and the videos from Google I/O are up (take a look at Steve Yegge's Server-side JavaScript on the Java Virtual Machine.) I'm a huge fan of TED - Sir Ken Robinson's talk on education has been on my iPhone since day one and is a much watch. Anyway, Glenn Vanderburg recently posted a link to a conversation about beautiful code recorded at JAOO. The 20-ish minute chat is full of lots of nuggets including a Thomas Aquinas reference. Software is much more art than engineering, something that is often lost of the non-technical and creativity is (rightfully) getting more attention these days. I found myself nodding along throughout, well worth a watch.
